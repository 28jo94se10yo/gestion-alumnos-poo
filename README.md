# Gestor de Alumnos y CRUD de Inscripciones

Este proyecto es un sistema de gestión académica integral desarrollado en **Python** utilizando el paradigma de **Programación Orientada a Objetos (POO)**. 

### 🚀 Funcionalidades Principales:
* **Gestión de Alumnos:** Permite el alta (con validación de legajos únicos), baja, modificación y listado de estudiantes.
* **Administración de Materias:** Registro de materias con su respectiva carga horaria semanal.
* **Sistema de Inscripciones:** Lógica para vincular alumnos con materias, validando la existencia de ambos y evitando duplicaciones.
* **Seguimiento Académico:** El sistema procesa notas parciales, calcula promedios y determina el estado final (Regular, Promocionado o Desaprobado).

### 🛠️ Conceptos Técnicos Aplicados:
* **Clases y Objetos:** Estructuración de datos mediante clases (`Alumno`, `Materia`, `Inscripcion`, `Fecha`).
* **Encapsulamiento:** Uso de métodos para modificar estados internos de los objetos de forma segura.
* **Lógica de Validación:** Control de errores en el ingreso de notas y códigos duplicados.
* **Interfaz de Consola:** Menú interactivo para una navegación fluida por todas las opciones.

### 💻 Cómo ejecutarlo:
1. Asegúrate de tener Python instalado en tu sistema.
2. Descarga el archivo `Gestor_De_Alumno.py`.
3. Ejecútalo desde la terminal con el comando: `python Gestor_De_Alumno.py`
