# Sistema de Gestión Bancaria Multicuenta

Un simulador robusto de backend bancario que permite administrar múltiples cuentas de forma simultánea, aplicando conceptos de escalabilidad y seguridad de datos.

### 📈 Operaciones Soportadas:
* **Gestión de Saldo:** Depósitos, extracciones y transferencias con validación de fondos.
* **Control Multicuenta:** Capacidad de manejar un gran volumen de cuentas mediante indexación eficiente.
* **Seguridad:** Validación de transacciones para prevenir errores de lógica financiera.

### 🛠️ Conceptos Técnicos Aplicados:
* **Programación Orientada a Objetos:** Clases para entidades `Cuenta` y `Banco`.
* **Lógica de Negocio:** Implementación de reglas financieras y validación de tipos de datos.
